# Szpiro–Fenchel: Resultados Computacionais
Repositório para experimentos numéricos (projeção–compressão–renormalização–estabilização) em 1^⊥, com foco em energia positiva, espectro e condicionamento.

## Como rodar
- Python >= 3.10
- pip install -r reproducibility/requirements.txt
- python scripts/run_all.py

## Dados
- Datasets grandes: publique no Zenodo (DOI aqui).
- Pequenos CSVs gerados em results/tables e data/processed.
