# Szpiro–Fenchel — Experimentos e Figuras

Repositório para scripts, notebooks e figuras relacionadas a espectros em ciclos \(I_n\), renormalização \(n^2\), e análises de \(\lambda_{\min}^+\), \(\lambda_{\max}\) e \(\kappa_{\mathrm{wh}}\).

## Estrutura
- `notebooks/`: cadernos exploratórios.
- `scripts/`: scripts reprodutíveis (CLI).
- `src/`: funções reutilizáveis.
- `data/`: dados brutos e processados.
- `figs/`: figuras (PNG/PDF).
- `tests/`: testes.
- `docs/`: documentação.

## Uso rápido
1. pip install -r requirements.txt
2. python scripts/generate_cycle_spectra_and_figs.py
